WHAT EXACTLY IS AN API?

An API is a service created by the host to enable easy implementation of specific features of the host into different projects. The service provided is standardized so that many types of projects can use it.

FOR EXAMPLE

The Windows API index is a list of Windows elements, like buttons, that developers can use in their application, so that it runs seamlessly on Windows machines. Here is the documentation

You can even use an API to give your android app a neural network! This makes it much easier for developers to implement this high end tech without having to make the whole network themselves. Here is the documentation.

WEB API'S ARE THE SAME

With a Web API different websites offer their services so that developers implement part of their websites. Those parts can be:

  • Data Collections
  • User information
  • Website Elements
  • Personal Account Control

The output from a web api is going to be organized data, like a JSON file or and XML file.

FOR EXAMPLE

The Flickr API can be used to recover image data, via specific filters and search terms, so that you can import Flickr's images into your own application. Here is the documentation
